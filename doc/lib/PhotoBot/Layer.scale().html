NodeBox Libraries

photobot.Layer.scale()

Syntax

canvas.layers[i].scale(w=1.0, h=1.0)

Description

Resizes layer i to the given width and height. When width w or height h is a floating-point number, scales percentual (0.5 means 50%), otherwise scales to the given size in pixels.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
canvas.layers[1].scale(200)
