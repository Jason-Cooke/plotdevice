Layer.overlay()

Syntax

canvas.layers[i].overlay()

Description

Sets the blend mode of layer i to overlay. Overlaying is a combination of the multiply() command and the screen() command. Light pixels are screened, dark pixels are multiplied.

Overlay works significantly slower than multiply and screen, because it is a blend mode native to PhotoBot and not of the underlying Python Imaging Library.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
i = canvas.layer("newspaper.jpg")
canvas.layers[i].overlay()
