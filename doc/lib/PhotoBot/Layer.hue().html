Layer.hue()

Syntax

canvas.layers[i].hue()

Description

Sets the blend mode of layer i to hue. Replaces the hue of the bottom layer with the hue of the hued layer, while retaining the bottom layer’s saturation and brightness.

Hue works significantly slower than other blend modes, because it is a blend mode native to PhotoBot and not of the underlying Python Imaging Library.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
i = canvas.fill((185,255,0))
canvas.layers[i].hue()
