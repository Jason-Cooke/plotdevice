Canvas.layers[]

Syntax

canvas.layers[index]

Description

The list of all layers in the canvas. Layers can be retrieved by index, or by name. Indices start at 1, so the lowest layer is canvas.layers[1], the layer above canvas.layers[2] and so on.

The bottom canvas.layers[0] is reserved for the transparent background and must remain untouched.

Example

Retrieves the layer named robot.

photobot = ximport("photobot")
canvas = photobot.canvas(100, 100)
canvas.layer("images/robot.jpg", name="robot")
print canvas.layers["robot"].index()
